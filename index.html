Judul

PERANCANGAN WEBSITE BUKU SAKU ONLINE DENGAN MENGGUNAKAN FRAMEWORK LARAVEL 
                 Studi Kasus: {'SMKN 1 Bandar Lampung'}

Latar Belakang

“ Dalam proses input data identitas, nilai etika serta sopan santun siswa berada di dalam buku khusus yang dinamakan Buku Saku Siswa ”

Identifikasi Masalah

belum berjalan maksimal karena masih menggunakan media kertas dan diolah secara manual, belum ada aplikasi secara online.

Batasan Masalah

  • Membuat aplikasi buku saku secara online

  • Aplikasi ini dibuat untuk guru dan siswa yang ada di SMK N 1 Bandar Lampung.

Rumusan Masalah

  • Bagaimana membuat aplikasi penginputan data nilai etika dan sopan santun siswa secara online SMKN 1 Bandar Lampung?

  • Bagaimana cara membuat aplikasi yang dapat digunakan oleh guru dan siswa/i secara mudah dan efektif ?

Tujuan Penelitian

  • Diharapkan dapat membantu penerapan website buku saku online dengan menggunakan framework Laravel di SMKN 1 Bandar Lampung.

  • Memudahkan para guru untuk melakukan pengisian buku saku secara online.

Cara Kerja Laravel

  • Blade Template
  • Routing
  • Controller
  • Authentication
  • E-mail Class SMTP
  • Laravel Premision
  • Query Builder and ORM

Kesimpulan

  • Website Buku Saku Online telah berhasil dibuat dan diharapkan mampu meningkatkan tingkat kedisiplinan siswa.

  • Website yang digunakan dibangun menggunakan Framework Laravel

Saran

  • Website ini terkadang melakukan loading yang cukup lama seiring dengan bertambahnya data, untuk kedepannya diharapkan mempercepat waktu loading.

  • Penulis berharap agar aplikasi ini kedepannya dapat dikembangkan ke platform mobile seperti android.
